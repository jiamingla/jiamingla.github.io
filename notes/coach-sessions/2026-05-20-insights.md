## 一句話脈絡

D1-D3 主軸是**自我審查的覺察**。
D4-D9 主軸是**有意識的執行 + 渴望管理**。

最關鍵的揭曉：**「讓位/不展示」對自己的真實成本，比 D3 答的更高得多**——D8 上午選項 C「鬆一口氣，沒讓 Head 把我當搬運工」揭曉，原來 D5 我以為能撐三個月沒被看見，實際撐不過 30 分鐘的「被當搬運工」感。

結論：**純 B 路線（默默累積）會崩潰，純 C 路線（持續推動）會被擠壓——可持續版本是 90/10**。

## 不能丟掉的金句

> ⭐⭐ **「不是退讓，是調整射程。」**（D5 寫過、D7-D8 在實踐——這六天的中心金句）

> ⭐ **「senior 不是沒有失落的人，是知道失落是選擇的副作用、然後依然能做對的選擇的人。」**

> ⭐ **「親近 ≠ 自貶。親近 = 對方感覺被理解。過度親近 = 對方感覺欠你。長期過度親近 = 被當成『好用』，不是『該被尊重』。」**

> ⭐ **「間奏的功能不是『爭取』，是『呼吸』——讓自己不會因為太久沒被看見而崩潰。」**

其他：
- 「我以為自己能撐三個月沒被看見，後來才知道我連 30 分鐘的『被當搬運工』都受不了」
- 「八個月醞釀」——credit / 具體方案 / 抱怨 vs 解方，三個失敗條件改善了哪幾個？
- 「不報備、只同步」——senior 跟平行同事的協作姿態
- 「Phase 1/2 改成『心裡帶著，被問才講』」——預備但不主動上桌
- 「真正的訊號是：你願不願意，在感到失落時，依然做對的選擇」
- 「把『內部效率問題』升級成『業務風險問題』」——三選項框架的關鍵升級
- 「漏掉的不是眼前正在測試的產品，而是 PM 沒發給我們的產品」（金句潛力高）

## 對 PDT Coach 累積反饋（v1.x 升級建議）

### 用了一週發現最有用的地方

1. **跨對話的記憶串聯**——「昨天你才剛聊過⋯⋯」這種連線是自己看不到的角度
2. **戳真實動機的問題**——「繼續推動是相信還是捨不得？」「你期待 Head 做什麼？」這種題自己問不出來
3. **檢驗題框架**——「三個月沒人發現你還會做嗎」這類題隔一段時間自己重答，答案會變
4. **行為層級的觀察**——不只給建議，還點出習慣性模式（自我審查的不同變形）

### v1.x 候選新增（從 5/15-5/20 累積）

- **第 12 面向：「渴望強度的真實版本 vs 我以為的版本」**——當使用者用「沒想法」「沒差」「無所謂」這種詞時，協助辨識是真的還是保護色
  - 候選提問：「如果這件事走向 A，你會鬆一口氣；走向 B，你會失落——比較一下哪個比較強？」
  - 這個面向是 D8 上午選項 C「鬆一口氣」揭曉的核心——**真實渴望強度跟自己以為的不一樣，是這六天最大的揭曉**

### Coach 已知限制（也記下來）

1. **時間感不準**——Coach 沒有真實時間軸感，容易把「昨天」當「前天」
   - 解法：對話開頭寫日期時間，事件加時間錨點
2. **單次對話量過大時，判斷準確度下降**——5/20 下午壓測會議前 Coach 給過一次錯建議（先建議安全版、知道八個月醞釀後改口進階版）
   - 解法：對話到一定長度時，Coach 應主動踩煞車

## 三條主線（六天累積整理）

### 主線一：自我審查模式的進化

| 階段 | 形式 |
|---|---|
| D1 | 有想法不講（怕越界） |
| D2 | 沒把握不問（怕不專業） |
| D3 | 用「成長」包裝「不甘心」 |
| D6 | 用「自我打折」換對方好答應 |
| **D8 上午** | **用「沒想法」當保護色** |

**D8 下午做了反向動作**——「八個月醞釀」誠實說出口。**這個自我審查鬆動了一層**。

→ 文章 D 的素材就在這條主線

### 主線二：渴望管理的成熟

| 時間 | 狀態 |
|---|---|
| D3 | 列出五個回報，核心是「被看見」 |
| D5 | 選 B 路線，答檢驗題「可能會猶豫」 |
| D7 | 選讓位（壓測會議用 B 版本開場），結束時失落 |
| **D8 上午** | **揭曉「鬆一口氣，沒讓 Head 把我當搬運工」** |
| **D8 下午** | **找到 90/10 節奏** |

→ 文章 A、F 的素材就在這條主線

### 主線三：路牌姿態的具體實踐

D7-D8 連續做了一系列 senior 動作：
- 約面談不在 Slack 拍板
- 邀請盟友 + 給具體理由
- B 版本開場讓對的人主導
- 不主動拿 Phase 1/2（聽 Head 的提醒）
- 用三選項框架交方案
- 把「內部效率問題」升級成「業務風險問題」
- 提案前同步盟友（不報備、不徵求）

→ 文章 B、C、E 的素材就在這條主線

## 「不要做」清單（這六天累積的禁忌）

從這六天累積出的禁忌（**未來寫文章時可以變成「給自己」的提醒清單**）：

1. 不要用強詞掩蓋不確定（「肯定」「應該」洩漏猶豫）
2. 不要用打折換對方舒適（時間 / 錯誤 / 判斷力打折）
3. 不要替別人決定他的時程/容量（不要幫 PM/RD 開脫）
4. 不要在同一週密集 surface（90/10 節奏關鍵在「間奏」）
5. 不要把每件事都當大事辦（後果嚴重 → 多想；不嚴重 → 直覺）
6. 不要把選擇 = 推開渴望（讓位後的失落要看見、要管理）

## 最深的提醒（給未來的自己）

> **D5 我以為自己能撐三個月沒被看見，D8 才知道我連 30 分鐘都受不了。**

這個揭曉不丟臉——丟臉的是 D5 答題時還不知道。

下次設定「我願意付什麼代價」這種題目時，先記得：**我以為我願意付的，跟我實際願意付的，可能差兩個量級**。

寫文章 F 時就是要保留這種誠實——**不是「我預測得很準」，是「我終於誠實了」**。
